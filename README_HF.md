# 🎵 Vocal Remover Pro - فصل الأصوات والآلات الموسيقية

تطبيق متقدم لفصل المسارات الصوتية (الأصوات، الطبول، الجيتار البيس، وباقي الآلات) باستخدام الذكاء الاصطناعي.

## ✨ الميزات

- 🚀 **سحابي 100%** - لا يحتاج تثبيت، يعمل مباشرة من المتصفح
- 📱 **متجاوب** - يعمل على الموبايل واللاب توب
- 🛑 **إلغاء فوري** - أوقف أي عملية بضغطة زر
- 🎯 **دقة عالية** - يستخدم نموذج HT-Demucs الأحدث من Facebook Research
- 🔍 **تشخيص شامل** - تقرير JSON جاهز لحل المشاكل

## 🎯 كيفية الاستخدام

1. **ارفع ملفك الصوتي** أو أدخل رابط URL مباشر
2. **اضغط "ابدأ فصل الملف"**
3. **انتظر النتيجة** وتابع التقدم عبر شريط الحالة
4. **حمّل الملفات المفصولة**:
   - 🎤 Vocals (الأصوات)
   - 🥁 Drums (الطبول)
   - 🎸 Bass (الجيتار البيس)
   - 🎹 Other (باقي الآلات)
