- [ ] حدّث `README.md` إذا لزم الأمر

## أثناء الإصدار

### 4. إنشاء Git Tag
```bash
git add .
git commit -m "Release v2.0.1: وصف موجز للتغييرات"
git tag v2.0.1
git push origin main --tags
```

### 5. إنشاء GitHub Release
1. اذهب إلى **Releases** → **Create a new release**
2. Tag version: `v2.0.1`
3. Release title: `Vocal Remover Pro v2.0.1`
4. وصف الإصدار (استخدم القالب أدناه)
5. ارفق ملف ZIP (اختياري)
6. اضغط **Publish release**
